> IT[<sup>[wiki]</sup>][wiki_it], a.k.a Information
> Technology，亦即「資訊技術」，主要用於管理和處理資訊所採用的各種技術縱稱。
>
> 該文章內容絕大部分擷取自網路，為多方資源整併、摘錄、提煉之產物。
> 同一條目可能參考多達十幾種資料來源，在此僅放上較有參考價值、且無干擾性廣告之來源。
